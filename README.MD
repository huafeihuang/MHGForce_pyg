

# Formulating and Representing Multiagent Systems With Hypergraphs in Pytorch

PyTorch implementation of MHGForce [1].

## Usage

#### train MHGforce

```shell
python train_MHGForce.py --dname cora --method HyperGCN --use_physic
```
